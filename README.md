# Simple Python Calculator

This repository contains a basic Python calculator that performs fundamental arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

## Requirements

- Python 3.x

## How to Run

1. Clone the repository

git clone https://github.com/your-username/repository-name.git

2. Go to the project folder

cd repository-name

3. Run the program

python app.py

## Example

Enter first number: 10  
Enter second number: 5  

Addition: 15  
Subtraction: 5  
Multiplication: 50  
Division: 2

app.py → Main calculator program
